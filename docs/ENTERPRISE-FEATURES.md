# Enterprise Features and Flow

## Core Enterprise Capabilities

1. Multi-tenant SaaS management
2. Role based access control
3. Client, site, post and post order management
4. Guard profile, documents, licenses and certifications
5. Shift scheduling, confirmation, no-show and conflict alerts
6. GPS/geofence attendance
7. QR/NFC patrol checkpoint verification
8. Incident reporting with photos/videos and approval flow
9. Daily activity report and handover
10. Dispatch/control-room live operations
11. Client portal with proof of service
12. Billing, timesheets and payroll foundation
13. Compliance, audit logs and expiry alerts
14. Analytics and SLA reporting

## Operational Flow

Company Admin -> Client -> Site -> Posts -> Post Orders -> Guards -> Schedule -> Attendance -> Patrol -> Reports -> Client Portal -> Billing/Payroll.

## Client Portal Proof

Clients should be able to see assigned guards, clock-in proof, patrol checkpoint scans, missed patrols, incidents, daily reports, and PDF exports.
